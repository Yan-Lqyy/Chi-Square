About This Application

This web application provides tools for performing common Chi-Square (χ²) statistical inference tests. It allows you to input your observed categorical data and perform the necessary calculations, including generating visualizations of the Chi-Square distribution and calculating effect sizes and post-hoc residuals where appropriate.

Use the navigation above to select the test you need or explore the resources.

Available Tools

  • Chi-Square Goodness of Fit Test: Test if the observed frequency distribution of a single categorical variable differs significantly from a hypothesized expected distribution.
  • Chi-Square Test for Contingency Tables: Test for a statistically significant association between two categorical variables (Test of Independence) or compare the distribution of a variable across different groups (Test of Homogeneity).
  • Chi-Square Critical Value Lookup: Find the critical value from the Chi-Square distribution for a given degrees of freedom (df) and significance level (alpha).
  • Learn More About Chi-Square Tests: Understand the theory, assumptions, and interpretation behind these tests.
